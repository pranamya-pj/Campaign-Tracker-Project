# Campaign Tracker
## Features
- Dummy login: Username:admin, Password:1234
- Add, delete, search campaigns
- Node.js + JSON backend
- Responsive UI

## Setup
1. Open VS Code and open the project folder 'CAMPAIGN-TRACKER'
2. Open terminal 
3. Run:
cd backend
npm install
npm start

4. Open  frontend/login.html' in browser or Open with live server and make sure backend is running, then open 'frontend/login.html'.
5. Use the form to add campaigns. The data is persisted in 'backend/campaigns.json'.
